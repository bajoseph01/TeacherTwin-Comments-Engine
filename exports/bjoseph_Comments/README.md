# B. Joseph Curated Snapshot

This folder is the repo-visible handoff surface for the current B. Joseph work.

Structure:

1. `final/`
   Current review-ready deliverables.
2. `batches/`
   Reusable per-batch assets such as prompts, structured marks, comments, and verification reports.
3. `reference/`
   Teacher-specific reference packs that are useful for future batches.

Raw extraction text and older root-level export clutter were moved into ignored `workspace/reference/` so the repo-facing tree stays focused on reusable or final artifacts.
